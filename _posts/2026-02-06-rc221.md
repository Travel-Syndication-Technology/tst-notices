## Hotel
- UI Improvements:
  - Date picker styling updated to match the Destination and Occupancy inputs for a more consistent search form appearance.
  - Mobile map view: Repositioned the "search this area" input below the draw-radius button to prevent overlap with zoom controls.
  - Mobile hotel details: Added a loading indicator to the bottom action bar while room rates are being fetched.
- Bugs Fixed:
  - Sabre-modified bookings now capture hotel chain information in the TST system and back-office integration exports.
  - VRBO properties are now being displayed for international locations.

  - TourBook Spotlight links now correctly open the new hotel details page.
