# Summary

The first release of 2026, RC219 contained mainly minor bug fixes and updates to ongoing conversion tests for various products

## Platform

### Production
- Trip Context is available for all clubs behind a feature flag.
- <b>Benefits to Agents:</b>
  - Provides context to the Agent's trip
  - Agents are always 2 clicks away from the cart or itinerary, checkout, and starting a new trip,
  - Agents can quickly switch between trips via the booking or trip details.
- Bug fixes and performance improvements

### Preview
- New Left Nav for Admin and Itinerary products
