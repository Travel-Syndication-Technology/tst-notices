# Summary

## Activity
- Fixed multi-ticket pricing inconsistencies with Pleasant Activities.
- Resiliency work to ensure that the activity product is handling errors well.

## Air
- Sabre Bugs Fixed:
  - Seats show status as 'pending ticketing'
  - Exchange did not sync back new pricing into TST
  - Flight failed to refresh - detect ticket exchanges when exchange status is in allCoupons
- Priceline Bugs Fixed: 
  - Baggage fees link shows all bags free
  - Error accessing Fare Rules

## Car
- Sabre Bugs Fixed:
  - Car Booking Failure
  - Sending CDP code for Budget car

## Cruise
- Reduce errors in cart:
  - Improved error handling for phone numbers received from CRMs that donâ€™t match our validation format.

## Guided Tour
- Fix Tour Contact-to-book form to use club affiliation.

## Hotel
- New Features & Enhancements:
    - Multi-Currency Support for Sabre: Sabre provider rates now display in both the licensee currency and the Hotel Provided currency when they differ. This feature is currently under a Feature toggle and will be enabled shortly after release.
    - Back-Office Integration: All GDS bookings are now marked as "Payment complete" to assist with back-office data synchronization.
- Sabre Bugs Fixed:
    - Duplicate Daily Rates: Fixed an issue where modifying a Sabre Hotel booking caused duplicate daily rates to appear.
    - Tax Calculation: Corrected an issue where specific tax lines were not added up correctly for Sabre rates.
    - Commission Capture: Fixed incorrect commission capture for certain Sabre bookings.
- General Hotel Bugs Fixed:
    - Timezone Booking Restriction: Removed a timezone factor that was preventing users in certain regions from shopping same-day hotel inventories.
    - Expedia Price Discrepancy: Resolved cases where the cart showed a higher price than the room selection page for certain Expedia Hotel Collect rates.
    - CUG pricing icon now displays correctly when exclusive rates are available.
    - Fixed an issue where pricing did not update properly after changing the search location.
    - Restored the AAA "Inspected Clean" certification badge on hotel cards.
    - Fixed the location search dropdown, which occasionally gets stuck in a loading state.
    - Fixed an issue where the "Remove all filters" button incorrectly switched to map view.

## Platform
- Analytics cross-reporting to GTM/GA4 (in limited testing)
- New Reservation Search form for Consumer bookings.

## Prepack
- Fixes for Softvoyage booking failures
- Refresh Pleasant bookings completed directly
- Softvoyage Remote IP fix
