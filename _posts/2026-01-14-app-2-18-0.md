## Overview
App 2.18.0 focuses on billing clarity, operational reliability, and stronger administrative visibility.

## Highlights
- Clearer billing pages with improved plan and payment history visibility.
- More resilient background processing through smarter retry behavior.
- Expanded audit events for key account and permission changes.

## Release notes
### Added
- Billing history with export support.
- Audit log entries for role changes and API key lifecycle events.

### Changed
- Background retries now use exponential backoff to reduce contention during partial outages.
- Settings navigation reorganized for faster access to common actions.

### Fixed
- Resolved an edge case where invoice PDFs could omit line-item totals.
- Reduced intermittent timeouts when loading large audit log pages.

## Compatibility and rollout
- No downtime expected.
- If you maintain internal documentation or runbooks that reference billing navigation, update screenshots as needed.

## Known issues
- Some older browsers may display a simplified billing chart style. A compatibility update is planned.
